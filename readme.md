# Proxy Search

Proxy Search is a self-hosted tool to search and obtain the fastest proxy.

## Features

- Parse free proxy websites.
- Support IPs locally.
- Test actual speed in MB/s.
- Self-hosted.

## Implementation

- Flask as backend and API.
- Pure HTML for the main interface.
- jQuery & JavaScript.
- Docker.
